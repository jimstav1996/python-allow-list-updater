# Python Allow List Updater

## Overview
This project demonstrates a Python algorithm that updates an IP allow list by removing unauthorized IP addresses.

## Scenario
An organization maintains an allow list of approved IP addresses. Some IP addresses must be removed because they should no longer have access. This Python script reads the allow list, removes matching IP addresses from a remove list, and writes the updated list back to the file.

## Tools Used
- Python
- Text file processing
- File handling with `open()`
- List methods

## Skills Demonstrated
- Reading files with Python
- Writing files with Python
